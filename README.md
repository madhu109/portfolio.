# Personal Portfolio Website

## Overview
This is my personal portfolio website showcasing my skills, projects, education, and contact information. The website is designed to provide recruiters, clients, and collaborators with an overview of my technical background and development experience.

## Features

- Responsive and modern design
- About Me section
- Skills and Technologies section
- Projects showcase
- Education details
- Contact information
- Social media and GitHub links
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap

### JusticeDesk – Legal Information and Consultation Platform
- Developed a web platform providing categorized legal information.

### Movie Explorer Application
- Developed a movie discovery platform using TMDB API.
- Implemented movie search and watchlist functionality.
